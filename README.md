# Leitura Dinamica

Leitura Dinamica e uma pagina simples para treinar leitura no navegador.

Voce pode colar um texto, escolher um modo de leitura e praticar de forma direta.

## O que voce pode fazer

Esta pagina possui 4 modos:

- `RSVP`: mostra uma palavra por vez
- `Foco`: mostra o texto completo com destaque na leitura
- `Chunking`: mostra grupos de palavras
- `Teste de Velocidade`: mede seu tempo de leitura e calcula seu PPM

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Escolha uma das abas da pagina.
3. Cole ou digite o texto no campo indicado.
4. Clique em `Iniciar`.
5. Use `Pausar`, `Parar` ou `Reiniciar` quando quiser.

## Como funciona cada modo

### RSVP

O texto aparece uma palavra por vez no centro da tela.

### Foco

O texto inteiro aparece na tela, mas uma palavra fica destacada por vez.

### Chunking

O texto aparece em blocos de palavras, em vez de aparecer palavra por palavra.

### Teste de Velocidade

1. Cole um texto completo.
2. Clique em `Iniciar`.
3. Leia no seu ritmo.
4. Quando terminar, clique em `Parar`.
5. Veja o resultado com:

- tempo total
- numero de palavras
- palavras por minuto (PPM)

## Dica de uso

Teste velocidades diferentes e veja qual delas fica mais confortavel para voce.

## Observacao

Esta pagina funciona direto no navegador e nao precisa de login nem instalacao.
